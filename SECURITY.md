# Security Policy

## Supported Versions

Following Versions of PBL (PersonalBlacklist) are supported by World of Warcraft GameClient Version 9.1.5 or higher:

| Version | Supported          |
| ------- | ------------------ |
| 3.7.x   | :white_check_mark: |
| 3.6.x   | :white_check_mark: |
| < 3.5   | :x:                |
